========================================
Visual Studio Code Extension for MongoDB
========================================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

MongoDB provides an extension for
`Visual Studio Code <https://code.visualstudio.com/>`__ which lets
you work with MongoDB and your data directly within your coding
environment.

.. figure:: /images/compass/vsce-landing.png
  :figwidth: 700px

You can use the Visual Studio Code extension to:

- :ref:`Connect to a MongoDB Cluster <vsce-connect>`.

- :ref:`Navigate your databases, collections, and documents
  <vsce-databases-collections>`.

- :ref:`Write queries and perform CRUD operations on your data
  <vsce-crud>`.

- :ref:`Write and execute aggregation pipelines on your data
  <vsce-aggregation>`.

To learn how to install the MongoDB Visual Studio Code Extension, see
:ref:`vsce-install`.
